# Smart-recipe-finder
Smart Recipe Finder is an AI-powered application that predicts cuisine types, validates ingredients, and generates unique recipes with nutrition, images, and voice instructions.
It also includes a weekly planner, PDF export, and podcast-style cooking assistant.
